Here’s a structured table with **components, alternatives, and their purposes**:

| **Component**          | **Alternative(s)**        | **Purpose** |
|------------------------|-------------------------|-------------|
| **ESP32**             | Arduino Uno, Raspberry Pi Zero W | Acts as the **central processing unit**, handling data collection, processing, and wireless transmission via **Wi-Fi & Bluetooth**. |
| **DHT22**             | DHT11, BME280, SHT31 | Measures **temperature and humidity**, which impact air quality and AQI calculations. |
| **PMS5003**           | SDS011, PMS7003, Nova PM Sensor | Detects **PM1.0, PM2.5, and PM10 particles**, providing crucial data for AQI assessment. |
| **Neo 6M GPS Module** | Ublox NEO-7M, Quectel L80 | Tracks **location coordinates** to map air quality levels in different areas. |
| **0.96-inch OLED Display** | LCD 16x2, TFT Display | Provides **real-time AQI readings** on the device itself for easy monitoring. |
| **Jumper Wires**      | Dupont Wires, Male-Female Connectors | Establishes **electrical connections** between components for seamless data transfer. |
| **Breadboard**        | Perfboard, Custom PCB | Used for **prototyping** and assembling circuit connections without soldering. |
| **Micro-USB Cable**   | USB-C Cable, Barrel Jack Adapter | Supplies **power to the ESP32** and enables firmware updates or serial data transfer. |

In case of using the alternative components, you have to take permission from Wizkit.
